フレックスボックス

概要

ある要素の中で、要素を並べ方を変えたり、間に余白を入れたり、指定した比率で要素を配置するためのレイアウト手法

用語

フレックスコンテナ
フレックスボックスの大枠となる要素。
主軸と交差軸という軸が設定され、主軸はデフォルトでは左から右、交差軸はそれに直行するように設定されている。
フレックスコンテナに設定した要素の子要素はこの主軸に沿って並べられる。
フレックスアイテム
フレックスコンテナ内で並べられる子要素。

プロパティ

用語 説明
display:flex; 要素をフレックスコンテナとして設定するためのプロパティ/値。
display:inline-flex; inline-blockのように、フレックスコンテナを横並びに配置できる右ようにするための値。
display:flex;の幅がブロックボックスのデフォルトである「100%」なのと同じく、
display:inline-flex;の幅は、インラインボックスのデフォルトである「中に入っている要素の幅」となる。
flex-direction 主軸の方向を決めるためのプロパティ。
flex-direction:row; 初期値。左から右に設定される。
flex-direction:row-reverse; 初期値の反対で、右から左に設定される。
flex-direction:column; 初期値から90度回転。上から下に設定される。
flex-direction:column-reverse; columnの反対。下から上に設定される。
justify-content 主軸方向で要素をどう揃えるかを指定するプロパティ
justify-content:flex-start; 主軸の開始地点に要素をそろえる。
justify-content:center; 主軸に対して中央に揃える。
justify-content:flex-end; 主軸の終了地点に揃える。
justify-content:space-between; 主軸の開始地点から終了地点まで、
要素の余白が等間隔になるように配置する。
align-items 交差軸方向にどう揃えるかを決めるプロパティ。
justify-cotentと同じように:flex-start;、center、:flex-end;の値がある。
flex-wrap フレックスアイテムの幅の合計がコンテナの幅を超えた時、どのように配置するかを指定するプロパティ。
フレックスボックスは、上記の場合デフォルトでコンテナに収まるようアイテムが縮む仕様となっている。
これをflex-wrap:wrap;とすることで、アイテムが折り返すように配置される。
なお、折り返して複数行になった場合、それぞれがFlex Lineと呼ばれる行単位で分割される。
Flex Line flex-wrap:wrap;で折り返した時アイテムが複数行になった場合、
それぞれがFlex Lineと呼ばれる行単位で分割される。
上述のjustify-contentやalign-itemsなどのプロパティはこのFlex Lineごとに適用される。
gap 要素間の隙間を定義できるプロパティ。
あくまで"要素と要素の間"の隙間を定義するもののため、
コンテナのサイズと綺麗に合わせる場合などにはcalc()と組み合わせて使用すると便利。
align-self aling-itemsのフレックスアイテム個別指定ver。
交差軸方向に対してアイテム個別に対してどう揃えるかを指定するプロパティ。
指定する値はalign-itemsと同様に:flex-start;、center、:flex-end;が使われる。
なお、主軸に対して個別指定したい場合はmarginプロパティで可能。
order フレックスアイテムの並び順は、基本的にはHTML上で書いた通りになるが、
orderを使うことで、CSS上のみで並び順を制御できる。
初期値は0であり、指定した数字が小さい順に表示される。
また、orderにはマイナスの値も指定可能。
flex フレックスアイテムの幅がコンテナの幅に満たない場合や超えた場合にどうするかを指定するプロパティ。
デフォルトのinitialでは、「広がらないが縮む」という指定をとり、
autoと指定すると「広がるし縮む」、noneの場合は「広がりも縮みもしない」という指定になる。
また、１以上の整数値を指定することもできる。
挙動はautoと同じだが、
複数アイテムのサイズを比率固定で指定したい場合、整数を利用する。

例)600px幅のコンテナにそれぞれ50px(1)、100px(2)、150px(3)の3つのアイテムがあり、
[ 1:2:3 ]の比率で指定したいとする
flex:auto;を指定した場合、余白の300pxを3つのアイテムに対し均等に割り振るため
それぞれ150px、200px、250pxとなり比率が一定でなくなる
一方、flexに整数を割り振る場合、元のアイテムの比率に関係なく、比率[ 1:2:3 ]であれば
[ 1 ]にしたいアイテムにflex:1;、[ 2 ]のアイテムにはflex:2;、[ 3 ]にはflex:3;と指定することで
常に[ 1:2:3 ]の比率で配分される。
